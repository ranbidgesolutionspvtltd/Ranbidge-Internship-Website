
# RANBIDGE Solutions Internship Portal

A modern, responsive internship portal built with React, TypeScript, and Tailwind CSS.

## Features

- 🔍 **Advanced Search**: Filter courses by category, title, instructor, and curriculum
- 📱 **Responsive Design**: Works seamlessly on desktop and mobile devices
- 🎯 **Auto-Navigation**: Smart course recommendations and enrollment form access
- 🌐 **Social Media Integration**: Connect with LinkedIn, Instagram, YouTube, and Facebook
- 🏢 **Professional Branding**: Consistent company identity throughout
- 📚 **Course Catalog**: Comprehensive internship programs with detailed information

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Preview production build:
   ```bash
   npm run preview
   ```

## Technologies Used

- **React 19** - Modern UI framework
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **Vite** - Fast build tool and dev server
- **Lucide React** - Beautiful icon library
